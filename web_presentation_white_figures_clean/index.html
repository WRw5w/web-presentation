03 / SCALING LAW

Qwen 系列随规模只有缓慢提升；真正明显的跃迁来自 DeepSeek 的 Thinking 模式。系统需要的是推理过程，而不只是更大的参数量。

04 / INFERENCE DYNAMICS

温度影响很小，
是否显式推理才是关键

温度从 0 到 1.5，Thinking 模式始终稳定在约 61%；关闭 CoT 后降到约 55.9%。因此 Pairwise 比较不能退化成第一印象投票。

设计结论：让模型解释“为什么 A 更适合当前任务”，再输出偏好。

05 / CALIBRATION

置信度越高，
比较结果越值得用于筛选

DeepSeek 在高置信区间达到约 77.4%，GPT-5.1 达到约 86.8%。

c ≥ 0.7模型能够评估
自己的准确性

06 / COMPLEXITY DISCRIMINATION

差异明显时容易判断，
细粒度强强对比仍然困难

复杂度差距扩大后准确率快速升高；差距小时仍在 55%–60% 附近。World Model 适合粗筛，不适合代替真实 evaluator。

权限边界：模型只分配预算，真实训练与验证保留最终裁决权。
